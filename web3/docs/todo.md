## 使用规则

- `✅` 表示已完成。
- `⬛️` 表示未完成。
- 后续每完成一项，需要把对应状态从 `⬛️` 改为 `✅`。
- 详细时间线和实现说明继续记录在 `docs/web3-trading-tool-progress.md`。
- Binance 后续接入必须严格按官方 Spot API 文档实现，文档地址：https://developers.binance.com/docs/binance-spot-api-docs
- Binance 真实下单不提前做，先完成公共行情、交易规则、K 线、WebSocket 和预览能力。
